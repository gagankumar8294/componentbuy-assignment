# Students Table – React CRUD Assignment

## 📌 Project Overview

This project is a **Students Management Table** built using **React.js**.
It demonstrates frontend CRUD operations (Create, Read, Update, Delete) using **React state and local data** without a backend.

The application allows users to add, edit, delete, and download student records in Excel format.

---

## Tech Stack

* React.js
* JavaScript (ES6)
* CSS3
* SheetJS (XLSX library)

---

## ✨ Features

### Student Table

* Displays a list of students
* Columns: Name, Email, Age, Actions

### Add Student

* Form to add new students
* Validation for required fields

### Edit Student

* Edit existing student details
* Form auto-fills with selected student data

### Delete Student

* Delete student with confirmation dialog

### Excel Download

* Export student data as an Excel file (.xlsx)

### Loading State

* Simulated loading animation when the app loads

### Success Messages

* Shows confirmation messages when:

  * Student added
  * Student updated
  * Student deleted

---

## ⚙️ Installation & Setup

1. Install dependencies

npm install

2. Run the project

npm start

---

## Excel Export

The project uses **SheetJS (xlsx)** to convert JSON data into an Excel file.

Library used:
https://www.npmjs.com/package/xlsx

---

## CRUD Operations Implemented

Create – Add new student
Read – Display students in table
Update – Edit existing student details
Delete – Remove student from table

All operations are handled using **React useState**.
